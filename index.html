I've fixed the issue where secondary headings were appearing white and invisible. The problem was that `.section-title` was using `var(--light)` (white) as its color, which blended into the white card background. Here's the corrected CSS: ```css .section-title { color: var(--dark); /* Changed from var(--light) to var(--dark) */ border-bottom: 1px solid var(--border-color); font-weight: 600; margin-bottom: 1.4em; padding-bottom: 0.5em; } ``` ALLChat
